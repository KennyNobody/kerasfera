Its a simple, fast, lightweight (~2kB) plugin for jQuery 1.7+
that allows to control bar view using css.

+ Supporting touch, wheel, middle button and any kind of scrolling
+ As fast as native
+ Weight incredible small
+ Tested with IE7+, mobile and modern browsers

Usage

To init custom-scroll, include files on your page, and init on desired elements

  1. <!-- Custom-scroll CSS -->

  2. <link rel="stylesheet" href="css/jquery.custom-scroll.css"> 

  3.  

  4. <!-- jQuery 1.7.0+ -->

  5. <script src="js/jquery.js"></script>

  6.  

  7. <!-- Custom-scroll JS -->

  8. <script src="js/jquery.custom-scroll.js"></script>
  1. //Custom-scroll init example

  2. $('#example').customScroll();

Options

There is some options you may need:

  1. // Full list of options with default values

  2. // All options are optional

  3. $('#example').customScroll({

  4.   prefix: 'custom-scroll_',

  5. 

  6.   /* vertical */

  7.   barMinHeight: 10,

  8.   offsetTop: 0,

  9.   offsetBottom: 0,

  10.   /* will be added to offsetBottom in case of horizontal scroll */

  11.   trackWidth: 10,

  12. 

  13.   /* horizontal */

  14.   barMinWidth: 10,

  15.   offsetLeft: 0,

  16.   offsetRight: 0,

  17.   /* will be added to offsetRight in case of vertical scroll */

  18.   trackHeight: 10,

  19. 

  20.   /* each bar will have custom-scroll_bar-x or y class */

  21.   barHtml: '<div />',

  22. 

  23.   /* both vertical or horizontal bar can be disabled */

  24.   vertical: true,

  25.   horizontal: true

  26. });

  27. 

  28. 

  29. // To recalculate bar position use same method without options

  30. $('#example').customScroll();

  31. 

  32. 

  33. // If you need to remove custom-scroll there is 'destroy' method

  34. $('#example').customScroll('destroy');

  35. 

  36. 

  37. // To change/get default options use

  38. $.fn.customScroll(someOptions);

How it works

After you init custom-scroll, it adds class to container, and makes new inner and bar
If there is already elements with those classes, they will be used

  1. <!-- before -->

  2. <div>

  3.   ...

  4. </div>

  5.  

  6. <!-- after: adds class -->

  7. <div class="custom-scroll_container">

  8.   

  9.   <!-- adds inner -->

  10.   <div class="custom-scroll_inner">

  11.     ...

  12.   </div>

  13.   

  14.   <!-- adds barHtml from options with class -->

  15.   <div class="custom-scroll_bar"></div>

  16.   <div class="custom-scroll_bar-x"></div>

  17. </div>

Hidden, max-height elements, paddings

1. Hidden element should be inited right after it will be shown. Since its not possible to properly count all dimensions of hidden elements

2. Max-height works fine, if you dont use horizontal scroll

3. Avoid using padding-top and padding-bottom for scrolling container. May have glitches.
